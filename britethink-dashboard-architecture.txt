12. URL ROUTING STRUCTURE
═══════════════════════════════════════════════════════════════

Base: /c/{course-slug}?currentEnrollmentId={enrollmentId}

Routes:
- Home:              /c/{slug}
- Assessment List:   /c/{slug}/assessment
- Assessment Detail: /c/{slug}/assessment/{assessmentId}
- Progress:          /c/{slug}/progress
- Portfolio:         /c/{slug}/portfolio
- Course Documents:  /c/{slug}/course-documents
- Personal Documents:/c/{slug}/personal-documents
- Materials:         /c/{slug}/materials
- Work Hours:        /c/{slug}/work-hours
- Members:           /c/{slug}/members

Query Parameters:
- currentEnrollmentId: always present (assessor's enrollment ID)
- learnerEnrlId: when learner filter is active
- unitId: when unit selected in progress
- outcomeId: when outcome selected in progress
- criterionId: when criterion selected in progress
